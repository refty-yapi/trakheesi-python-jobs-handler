# CLAUDE.md

This file provides guidance to Claude Code when working with code in this directory.

## Commit Guidelines

- **Do NOT add Co-Authored-By**: Never add "Co-Authored-By: Claude" or similar attribution to commits or PRs

## Pull Request Guidelines

- **Always link PRs to tickets**: Include "Works on https://github.com/refty-yapi/project-tickets/issues/{number}" in PR descriptions
- **Ask for ticket number**: If the ticket number is not known, ask the user before creating a PR
